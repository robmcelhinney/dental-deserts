# Scope Definition

## MVP

- Postcode search and map centering.
- Nearest practices sorted by distance.
- Adult/children acceptance flags (`yes`/`no`/`unknown`).
- Persistent caveats next to results.
- Reproducible local data build command.

## V1

- Radius filters (1km, 2km, 5km, 10km).
- Adult vs children primary toggle.
- Dental-desert message: no adult (or child) acceptance within selected radius.
- Area choropleth and comparison card.

## V2

- Travel-time / isochrone mode.
- Monthly snapshot storage and trend views.
- Optional API endpoint with indexed nearby query.
- Change alerts and data export options if licensing permits.
